# Use Cases

### Niet ingelogd gebruiker
<ol>
    <li>[X] Als gebruiker moet ik een account kunnen aanmaken.</li>
    <li>[X] Als gebruiker moet ik met mijn account kunnen inloggen.</li>
    <li>[X] Als gebruiker kan ik alle producten zien die te koop zijn.</li>
    <li>[ ] Als gebruiker kan ik een op een product klikken om het specifieke product te bekijken.</li>
    <li>[ ] Als gebruiker kan ik filteren in de lijst met alle producten op basis van category.</li>
    <li>[X] Als gebruiker kan ik op het producten overzicht pagina een product toevoegen aan een winkelmand.</li>
    <li>[X] Als gebruiker kan ik op een specifieke product pagina een product toevoegen aan een winkelmand.</li>
    <li>[ ] Als gebruiker kan ik zoeken op een specifiek product op basis van een zoekbalk.</li>
    <li>[X] Als gebruiker kan ik op de winkelwagen klikken om mijn winkelwagen te bekijken.</li>
    <li>[X] Als gebruiker kan ik in mijn winkelwagen het aantal keer dat ik een specifiek product wil hebben verhogen of verlagen.</li>
    <li>[X] Als gebruiker kan ik in mijn winkelwagen een specifiek product verwijderen uit mijn winkelwagen.</li>
</ol>
